# 📚 Gabriel Ensina | Matemática e Física

Site institucional desenvolvido para o professor particular Gabriel Duque, focado em apresentar sua trajetória como estudante de Física na **UFMG** e captar novos alunos para aulas de reforço e preparatórios.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído de forma limpa e eficiente utilizando:

* **HTML5** para a estrutura das páginas.
* **CSS3** personalizado para estilização e efeitos.
* **Bootstrap 5** para garantir um layout responsivo e moderno.
* **JavaScript** para componentes dinâmicos.
* **Font Awesome** para os ícones interativos.

## 📱 Funcionalidades

* **Design Responsivo:** O site se adapta perfeitamente a computadores, tablets e celulares.
* **Seções Dinâmicas:** Cabeçalho, rodapé, botões de chamada para ação (CTA) e botão flutuante de WhatsApp injetados via JavaScript para facilitar a manutenção.
* **Integração com WhatsApp:** Link direto para agendamento de aulas experimentais.
* **Metodologia Detalhada:** Seção dedicada a explicar o diferencial didático do professor.
* **Provas Sociais:** Espaço para depoimentos de alunos reais.

## 📁 Estrutura do Projeto

Para manter a simplicidade e a velocidade de carregamento, os arquivos principais estão organizados na raiz do projeto:

* `index.html` - Página principal (Home).
* `sobre.html` - Trajetória e formação do Gabriel.
* `metodologia.html` - Detalhes sobre como as aulas funcionam.
* `contato.html` - Canais de atendimento e informações de horários.
* `styles.css` - Estilização personalizada e variáveis de cores.
* `script.js` - Lógica do site e componentes injetados.
* `/images` - Pasta contendo fotos e logotipos do projeto.

## 🎨 Cores e Identidade

O projeto utiliza uma paleta de cores que transmite confiança e clareza, com tons de azul e amarelo, focados na legibilidade e na experiência do usuário.
